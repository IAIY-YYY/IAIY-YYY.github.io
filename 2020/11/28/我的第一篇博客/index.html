我的第一篇博客

2020.11.28 John Doe  热度 ℃

博客搭建(以下命令都是在cmd中执行)
• 下载安装环境变量(前往Hexo官网)
cmd(以管理员身份打开) 中输入 node -v 和 git -version 查看是否安装了环境。
• 安装Hexo
cmd(以管理员身份打开) 输入 npm install -g -hexo -cli
• 开始建立网站
用 hexo init XXX(自定义文件名) 来生产我们所需的文件。这时 Hexo 就会自动下载我们所需的文件到 XXX 中
如果出现错误则回到 XXX 中输入运行 npm install 然后就可以了
• 生产网站页面(生成public目录)
cmd(以管理员身份打开) 进入 hexo 自定义的位置(XXX) 输入 hexo generator，缩写输入运行 hexo g
如果出现错误(信息缺失)，则重新输入一遍即可。
• 运行网站
cmd(以管理员身份打开) 进入 hexo 自定义的位置(XXX) 输入 hexo server，缩写输入运行 hexo s
• 创建一篇文章(创建的文件在source文件中)
cmd(以管理员身份打开) 进入 hexo 自定义的位置(XXX) 输入 hexo new “XXX”
当每修改文件或者配置时，需要重新生成一下hexo。首先清除一下(hexo cls)，其次重新生成(hexo g)，然后启动服务(hexo s)
部署到Github公开使用
• 创建仓库
首先在自己的Github创建一个仓库。名字命名要和用户名相同，创建的新仓库(repositories)是空的

• 安装插件
cmd(以管理员身份打开) 进入 hexo 自定义的位置(XXX) 输入 npm install hexo-deployer-git -save
• 设置配置文件(_config.yml)
repo: 链接从Github空仓库中获取URL地址
• 部署开始
cmd(以管理员身份打开) 进入 hexo 自定义的位置(XXX) 输入hexo d 进行部署
如出现错误，要注意设置 Git 的 Username 和 Password ——要和Github账号一致
成功部署到Github中！！
博客主题设置
• 找到主题链接并下载
cmd(以管理员身份打开) 进入 hexo 自定义的位置(XXX) 输入 git clone 主题链接.git themes/主题名称
并在 XXX 中创建了 themes(主题存放目录) 目录
例如：git clone https://github.com/littem/hexo-themes-yilia.git themes/yilia
• 配置到博客中
• 配合完成后
当每修改文件或者配置时，需要重新生成一下hexo。首先清除一下(hexo cls)，其次重新生成(hexo g)，然后启动服务(hexo s)
